# AksharaShift - Complete Character Mapping Reference

## Overview

This document provides comprehensive character-to-character mappings for both ML and FML legacy font formats supported by AksharaShift.

## Unicode Malayalam Character Range

Malayalam characters in Unicode occupy the range **U+0D00 to U+0D7F** (3328-3455 decimal).

### All Other Consonants - FML Format

FML uses identical mappings as ML for:
- ക, ഖ, ഗ, ഘ (velar series)
- ച, ഛ (palatal stops)
- ട, ഠ, ഡ, ഢ (retroflex series)

- യ, ര, ല, ഴ, വ (semi-vowels)
- ശ (sibilant)

## Implementation Notes

### Character Processing Order

1. **Conjuncts first** (two-character combinations)
   - Check if next two characters form a valid conjunct
   - If yes, output the conjunct mapping and skip 2 characters
   
2. **Single characters** (if not a conjunct)
   - Output the character mapping
   - Skip 1 character

3. **Unmapped characters**
   - Pass through unchanged (English, numbers, other Unicode)

### Virama Handling

The virama character (്, U+0D3C) indicates consonant clusters:
- Typically produces no output (shown as empty string in mapping)
- Represents "half consonant" or consonant killer
- Followed by another consonant in conjuncts

### Edge Cases

**Empty Mappings**: Some characters (like virama) have empty string mappings - they produce no output

**Character Variations**: Some characters have multiple visual representations but same Unicode value

**Legacy Font Limitations**: Not all Unicode characters may be displayable in legacy fonts
